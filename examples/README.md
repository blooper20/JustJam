## üìö Tutorials

### 1. Custom Tuning

Learn how to use alternative guitar tunings like Drop D, DADGAD, etc.

**[‚Üí See Custom Tuning Guide](./custom-tuning/README.md)**

**Quick Example:**
```python
from src.tab_generator import TabGenerator

# Drop D tuning (DADGBE)
generator = TabGenerator(
    tuning=['D2', 'A2', 'D3', 'G3', 'B3', 'E4'],
    bpm=140
)

tab = generator.generate_ascii_tab(notes)
```

### 2. Batch Processing

Process multiple audio files at once and save results.

**[‚Üí See Batch Processing Guide](./batch-processing/README.md)**

**Quick Example:**
```python
import glob
from src.transcriber import transcribe_audio
from src.tab_generator import create_tab

for audio_file in glob.glob("resource/*.mp3"):
    notes, bpm = transcribe_audio(audio_file)
    tab = create_tab(notes, bpm=bpm)

    # Save to file
    output = audio_file.replace('.mp3', '_tab.txt')
    with open(output, 'w') as f:
        f.write(tab)
```

### 3. Section-by-Section Analysis

Analyze specific parts of a song (intro, chorus, solo, etc.)

**Example: Analyze only the chorus**
```python
from src.transcriber import transcribe_audio
from src.tab_generator import create_tab

# Chorus starts at 1:20 (80 seconds) and lasts 30 seconds
notes, bpm = transcribe_audio(
    "resource/song.mp3",
    start_offset=80.0,
    duration=30.0
)

tab = create_tab(notes, bpm=bpm)
print(tab)
```

## üí° Tips for Best Results

### 1. Audio Quality

**Good:**
- Clean guitar recordings
- Minimal background noise
- Solo guitar or guitar-prominent tracks
- Clear, uncompressed audio (WAV, FLAC)

**Challenging:**
- Heavy distortion/effects
- Multiple instruments playing simultaneously
- Compressed/low-quality MP3s
- Bass-heavy mixes that drown out guitar

### 2. File Preparation

```bash
# Convert video to audio (if needed)
ffmpeg -i video.mp4 -vn -acodec mp3 output.mp3

# Extract specific section
ffmpeg -i song.mp3 -ss 00:01:20 -t 00:00:30 chorus.mp3

# Normalize audio levels
ffmpeg -i input.mp3 -af "volume=2.0" output.mp3
```

### 3. Accuracy Tips

- **Use section-by-section analysis** for complex songs
- **Manually set BPM** if auto-detection is off
- **Try different start offsets** for the best part to analyze
- **Check the chord progression** - it helps verify accuracy

## üêõ Troubleshooting

### Issue: Few notes detected

**Solution:**
- Increase audio volume
- Use cleaner recording
- Try analyzing a different section

### Issue: Wrong BPM

**Solution:**
```python
# Override BPM manually
notes, _ = transcribe_audio("song.mp3")
tab = create_tab(notes, bpm=120)  # Force BPM to 120
```

### Issue: Unplayable fingerings

**Solution:**
- The algorithm prioritizes open chords (0-5 fret)
- For higher fret songs, consider capo position
- Use `tweak_tab_fingering` tool to adjust specific notes
